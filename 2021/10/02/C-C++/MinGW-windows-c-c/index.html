概要

MinGW (Minimalist GNU/GCC for Windows) 是专为Win32移植的产物
换言之，Linux C才是主流！

Windows MinGW环境搭建

首先安利一波MSYS2(Minial SYStem 2 )
它集成了一系列标准Linux工具，当然包含clang和gcc;
其实有很多类似的Shell: Cygwin, WSL...

msys2.org给出了详细的安装指南~

gcc/g++ manuals

这不禁让我想起大学刚接触linux c的开发情景；
gcc --help : 多使用help or Linux Manul 😄

Syntax

gcc [-c|-S|-E] [-std=standard] [-Wall] [-o <file>]

-c: Compile and assemble, not link ★
-S: Compile only
-E: Preprocess only, not compile, assemable or link
-std: standard, c11, c20 ★
-Wall: treat all Warnings as Error ★
-o: output filename
-O0: Optimization, watch this: gcc -o / -O options ★

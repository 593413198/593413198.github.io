提要

此规划写于2021年国庆
希望在三年后，能够达到如下的水平

1. 精通2+ 脚本语言

  • python 2/3
  • Lua
  • TypeScript
  • JavaScript

2. 精通1+ Strongly typed语言

  • C++
  • C
  • C#
  • Java

3. 实现3+ 大型GamePlay模块

  • 技能框架 Skill
  • xx战斗模块
  • UI模块 cocos…
  • FPS/TPS 射击模块
  • 商城模块 （偏server）
  • 时装外观模块
  • 匹配、排行模块 （偏server+database)
  • …

4. 熟悉1+ physics engine

  • box2d (2d)
  • physx (3d)
  • bullet (3d)

5. 手写一套opengl渲染框架

  • lighing manager
  • shadaw
  • texture
  • sader language
  • …

6. 手写一套粗略的3d engine

  • memory malloc
  • object manager (scene, model…)
  • model (fbx)
  • texture and material
  • shader
  • rendering pipeline
  • physics (how to give better api)
  • animation
  • editor (model and scene editor?)
  • asset manager and tools … (like unity?)

7. 精通Graphics

  • 良好的数学基础
  • 足够的coding实践
  • UE4/u3d的code learning

8. 持续不断的English Learning

  • Reading Paper (GDC…)
  • Reading Document (API…)
  • Maybe talking

9. 优秀的编程习惯Good Coding

  • Code review
  • Blog record
  • Learn good open-source code

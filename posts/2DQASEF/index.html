概要

CPU、内存、寄存器、指令集，这些与编译器的实现息息相关
本文只讨论编译背景下的内容

warning

阅读文档是最快速、最准确的信息渠道
以下只是一个自我学习和记录的过程

Memory: 内存

一段C代码进程，会将内存分为如下几个部分:

Memory Usage Symbol Example
code data .text int main() { // ... }
initialized data .data int8_t var = 0xff;
uninitialized data .bss int8_t var;
function call data stack
malloc (dynamically allocated) heap char *p = malloc(1024);

Register: 寄存器

  • Register is a very fast computer memory, used to store data/instruction in-execution;

阅读如下文档:

  • Computer Registers

  • Types of CPU Registers and Their Functions

Register Symbol function
Program Counter PC 指向要执行的指令
Accumulator AX 存一些运算结果
Stack Pointer SP 永远指向栈顶
Base Pointer BP 通用指针寄存器

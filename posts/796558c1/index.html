为什么选择hexo?

  • Opensource
  • Simple and Fast
  • Good Plugins and Themes

感谢Hexo，这是本博客的技术框架

开始hexo

这是官网的教程，简略但清晰：
Hexo Start

补充几点：

    1. 关注version，对dependencies问题引起足够重视
      npm list --depth 0 : 查看已安装的版本号
      npm audit fix --force : 强制安装所有依赖项
    1. 只需关注极少的几个指令
      hexo init [folder] : 初始化你的hexo目录
      npm install : 自动安装（读取package.json）
      npm new "you title" : 新建一篇文章 （它在你的/source/_posts下）
    • 如何删除一个blog？去_posts下删除对应md即可
      npm g : (generate)生成本地静态文件/public
      npm s : (server)启动本地服务器，供预览 localhost:4000
      npm d : (deploy)发布
      npm clean : 清除本地缓存与/public

hexo发布github

这里记录几个坑

  1. _config.yml中deploy使用git，对应repo应该是:
    git@github.com:{user_name}/{respository_name}git
  2. 添加github的SSH Keys
    ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
    cat ~/.ssh/id_rsa.pub
    github - setting - SSH adn GPG Keys
